Eclipse 3.2 requires Mac OS X 10.4 (Tiger) or higher and Java 1.4.2 or higher.
Since M5 Eclipse supports the Intel architecture.

Please use BOMArchiveHelper if StuffIt fails to extract the downloaded tar.gz
(https://bugs.eclipse.org/bugs/show_bug.cgi?id=131650).

In order to use the SWT_AWT bridge, OS X jre version 1.5.0 Release 4
(or greater) must be used, and the additional "SWT Compatibility Libraries"
must be installed.  These libraries can be downloaded from the Downloads
section of Apple Developer Connection (http://connect.apple.com).  It's free
to create an account at Apple Developer Connection to access the download
(180K).  (Eclipse report: https://bugs.eclipse.org/bugs/show_bug.cgi?id=67384).
