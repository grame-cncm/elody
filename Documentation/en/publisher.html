Publisher window

A central server located at java.grame.fr is available for Elody users to share documents. The publisher allows to publish and fetch documents. Elody documents contain a title, the document's author name, a short description of the content and the expression itself.

The Connect button starts a new connection between the Elody server and the user Elody environment. The list of all available documents is received and is displayed in a pop-up menu. To fetch a document, one just have to select a document name and press the Fetch button. The document title, author, description will be displayed and the document content will be available in the top box.

To publish a new document, one must first drop the Elody expression in the top box, fill the title, author and description zone and press the Publish button. After being received and saved by the server, the new document list will be sent to all connected users.
