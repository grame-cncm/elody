The Elody package

The Elody package contains:

  • A Java application: the Elody environment which allows to build, manipulate, load and save musical objects.
  • A documentation in HTML format
  • Several musical examples

    • Elody for the Macintosh

    MidiShare must be installed on your machine. It is part of the Elody Installer.

    Download Elody
     Europe Site  North America Site

    Elody works with the Apple MJR 2.0. The MRJ 2.0 must be installed before Elody can be used. Note that the MRJ 2.0 is installed with the Mac OS 8.0 and 8.1.

    • Use the installer to install Elody.
    • To launch Elody, double-click on the Elody application. The Elody.ini is a setup file used by Elody to open the different windows. The main Elody window contains several buttons which allow to open different Elody constructors.

    Elody mailing list

    The Elody mailing list is the way to share information between Elody users. It is also the way to find support for all your problems when using Elody.

    If you want to subscribe to the Elody mailing list, send the following:
    "subscribe"
    or from another account:
    "subscribe <your email adress>"
    in the body of a message to elody-users-request@grame.fr

    You shoud receive an email confirming your subscription.

    All messages to the list are archived. The archives are available via anonymous FTP, or via Majordomo:

    • Via anonymous FTP: at ftp://ftp.grame.fr/pub/Elody/archive/
    • Via Majordomo: using the "index elody-users" and "get elody-users" elody-users.YYMM" commands (send "help" in the body of a message to "Majordomo@grame.fr" for more info)

    The archives are broken down by month, and are stored in files named "elody-users.YYMM" or where "YY" is the year and "MM" the month. They are updated every night at 2am local time.

    For further information, directly contact the list owner.
