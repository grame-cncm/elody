The Elody package

The Elody package contains:

  • A Java application: the Elody environment which allows to build, manipulate, load and save musical objects.
  • A documentation in HTML format
  • Several musical examples

    Elody for Windows XP

    MidiShare must be installed on your machine. It is part of the Elody Installer.

    Download Elody
     Europe Site

    The Elody installer contains and install the Sun JRE 1.1.7.

    • Use the installer to install Elody
    • To launch Elody, double-click on the Elody file. The Elody.ini is a setup file used by Elody. The main Elody window contains several buttons which allow to open different Elody constructors.
