Couleurs


Les objets de base comme les notes ou les silences ont leur caractéristiques habituelles : la hauteur, la vélocité, la durée, le canal MIDI et un attribut supplémentaire : une couleur. La case de droite (port MIDI) contient le flot temps réel.

Les couleurs sont utilisées pour distinguer des notes ayant leur autres attributs identiques lors du processus d'abstraction. Lorsqu'on abstrait une note (en utilisant le constructeur Lambda) dans une expression musicale contenant des notes de différentes couleurs, seules les notes ayant la même couleur que la note argument seront prises en compte lors du processus d'abstraction.

Pour changer la couleur courante du constructeur Accords, il suffit de glisser la couleur désirée sur la fenêtre du constructeur Accords.
