L'archive Elody

L'archive Elody contient :

  • Une application Java : l'environnement Elody qui permet de construire, manipuler, sauver et restaurer des objets musicaux.
  • Une documentation au format HTML.
  • Plusieurs exemples musicaux.

Elody pour Windows XP

MidiShare doit être installé sur votre machine. Il fait partie de l'installeur d'Elody.

Télécharger Elody
 Site Europe

L'installeur d'Elody contient et installe le JRE 1.1.7 de Sun.

  • Utiliser l'installeur pour installer Elody.
  • Pour lancer Elody, double-cliquer sur le fichier Elody.bat. Le fichier Elody.ini est un fichier de configuration utilisé par Elody. La fenêtre principale contient plusieurs boutons qui permettent d'ouvrir les différents constructeurs.
