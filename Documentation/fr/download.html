L'archive Elody

L'archive Elody contient :

  • Une application Java : l'environnement Elody qui permet de construire, manipuler, sauver et restaurer des objets musicaux.
  • Une documentation au format HTML.
  • Plusieurs exemples musicaux.

Elody pour Macintosh

MidiShare doit être installé sur votre machine. Il fait partie de l'installeur d'Elody.

Télécharger Elody
 Site Europe  Site Amérique du Nord

Elody fonctionne avec le MJR 2.0.Apple. Le MRJ 2.0 doit être installé avant qu'Elody puisse être utilisé. Le MRJ 2.0 est installé avec le système Mac OS 8.0 et 8.1.

  • Utiliser l'installeur pour installer Elody.
  • Pour lancer Elody, double-cliquer sur l'application Elody. Le fichier Elody.ini est un fichier de configuration utilisé par Elody. La fenêtre principale contient plusieurs boutons qui permettent d'ouvrir les différents constructeurs.

Mailing liste Elody

La mailing liste Elody est le canal de partage de l'information entre les utilisateurs d'Elody. C'est également le lieu où vous pourrez trouver un support pour vos problèmes d'utilisation.

Si vous désirez souscrire à la mailing liste Elody, envoyez la commande suivante :
« subscribe »
ou bien, à partir d'un autre compte email:
« subscribe <votre adresse email> »
dans le corps d'un message adressé à elody-users-request@grame.fr

Vous devriez recevoir un email de confirmation.

Tous les messages de la mailing liste sont archivés. Ces archives sont accessibles via ftp ou Majordomo :

  • Via FTP anonyme : ftp://ftp.grame.fr/pub/Elody/archive/
  • Via Majordomo : grâce aux commandes « index elody-users » et « get elody-users elody-users.YYMM » (envoyez une commande « help » dans le corps d'un message à « Majordomo@grame.fr » pour plus d'information)

Les archives sont stockées par mois dans des fichiers séparés nommés « elody-users.YYMM » ou « YY » correspond à l'année et « MM » au mois. Elles sont mises à jour toutes les nuits à 2h heure locale.

Pour plus d'information, contactez directement le gestionnaire de la liste.
