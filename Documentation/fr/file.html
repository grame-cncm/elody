Gestionnaire de fichiers

Le gestionnaire de fichiers permet de sauvegarder et restaurer des expressions Elody sous différents formats. Lors d'une sauvegarde, le format de fichier choisi doit être spécifié dans le menu déroulant. Une autre manière de sauvegarder des expressions Elody est d'utiliser la fenêtre Document. Le bouton Chercher permet de charger une expression à partir de son URL.

Format HTML

Les documents Elody sont sauvés au format HTML. Il est ainsi possible de visualiser des expressions Elody avec un navigateur tel que Mozilla Firefox par exemple. L'extension .html du fichier doit être explicitement ajoutée pour qu'ensuite Elody puisse reconnaître et charger les fichiers.

Format MIDIFile

Elody peut lire et écrire des fichiers au format MIDIFile 0 et 1. Lorqu'un fichier MIDIFile est lu dans Elody, seules les notes sont prises en compte. Les autres types d'événements (contrôleurs, changement de programme, etc.) sont actuellement ignorés. Les expressions Elody sont sauvegardées au format 0 (une piste). L'extension .mid du fichier doit être explicitement ajoutée pour qu'ensuite Elody puisse reconnaître et charger les fichiers.

Format GUIDO

GUIDO est un format de description de partitions en notation traditionnelle. C'est un format textuel, facilement lisible et indépendant de la plateforme qui est capable de prendre en compte toutes les informations contenues dans une partition musicale. Le format GUIDO est développé par Holger H. Hoos et son équipe à l'université de Technologie de Darmstadt en Allemagne.

Elody peut lire et écrire des expressions au format Basic GUIDO.
