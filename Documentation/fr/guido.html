Afficheur GUIDO

GUIDO est un format de description de partitions en notation traditionnelle. C'est un format textuel, facilement lisible et indépendant de la plateforme qui est capable de prendre en compte toutes les informations contenues dans une partition musicale. Le format GUIDO est développé par Holger H. Hoos et son équipe à l'université de Technologie de Darmstadt en Allemagne.

Les expressions musicales peuvent être affichées en notation musicale en utilisant le GUIDO Note Server. Les expressions Elody sont converties au format GUIDO, le fichier obtenu est envoyé au GUIDO NoteServer qui retourne une image GIF qui sera affichée dans l'applet. Un facteur de zoom permet d'ajuster la taille de l'image.

Remarque : la conversion d'objets Elody au format GUIDO est un projet en cours de développement. Le résultat n'est pas toujours satisfaisant en particulier dans le cas le l'affichage de longues expressions. Nous travaillons avec les développeurs de GUIDO à l'amélioration des résultats.
